Usage: astral project [projectName]

This creates the basic file structure I use to work with Meteor.

```
client/
  helpers/
  stylesheets/
  views/
    application/
      layout.html.example

  main.html
  main.js

collections/

lib/
  router.js

public/
server/
  fixtures.js
  publications.js

```

To enable iron-router, astral project [projectName] -r
